# LingoMatch 🌍

**LingoMatch**, dil öğrenmeyi eğlenceli hale getiren bir web uygulamasıdır. Kullanıcılar farklı kültürlerden insanlarla eşleşerek dil pratiği yapabilir, kelime kartlarıyla hazinelerini geliştirebilir ve günlük görevlerle XP kazanabilir.

## 🚀 Özellikler

- **Onboarding Wizard** — Ana dil, hedef dil, seviye ve hobilerini seç
- **Keşfet Feed'i** — Canlı sosyal akış ve günlük görevler
- **Kelime Hazinem** — İnteraktif çevrilebilir flashcard'lar
- **Eşleşmeler & Sohbet** — Dil ortaklarıyla anlık mesajlaşma ve GIF sticker'lar
- **Profil & Rozetler** — XP, seri ateşi ve kazanılan rozetler
- **Karanlık / Aydınlık Tema** — Tam tema desteği

## 🛠️ Teknolojiler

- **HTML5** — Semantik yapı
- **CSS3** — Vanilla CSS, CSS Variables, glassmorphism, animasyonlar
- **Vanilla JavaScript** — Sıfır bağımlılık, tam işlevsel SPA

## 📂 Proje Yapısı

```
lingomatch1/
```

Herhangi bir build adımı gerekmez. `index.html` dosyasını tarayıcıda aç ya da bir local server kullan:

```bash

python3 -m http.server 3000

```
